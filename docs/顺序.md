完整调用链一图总结
config.py                    model.py
─────────                    ────────
rope_theta ─────┐
rope_scale_factor ──┐       NovaModel.__init__
                    │            │
                    └──→ precompute_rope_freqs()  ← 第三站：算旋转角度表
                              │                      （位置插值在这里：t = t / scale_factor）
                              ▼
                         self.freqs_cis  ← buffer，shape [max_seq_len, head_dim//2]
                              │
                    NovaModel.forward
                              │
                         freqs_cis[:seq_len]  ← 第四站：截取当前长度
                              │
                    TransformerBlock.forward   ← 第五站：透传
                              │
                    MultiHeadAttention.forward
                              │
                         apply_rotary_emb(q, k, freqs_cis)  ← 第六/七站：旋转 Q 和 K
                              │
                         Q·K^T / √d → mask → softmax → ×V  ← 正常 Attention 继续
简单来说：预算一次，每层用一次。位置插值的代码只有 precompute_rope_freqs 里的一行 t = t / scale_factor，它把位置索引压缩回训练范围，让模型在推理时能处理比训练更长的上下文。
